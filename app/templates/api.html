API ElasticCafe

A ElasticCafe possui uma API que permite aos desenvolvedores gerenciar pedidos de café de seus clientes. Abaixo, estão listados os principais endpoints da API:

Adicionar um pedido

Para adicionar um pedido, faça uma solicitação HTTP POST para o seguinte endpoint:

POST /add_coffee

O corpo da solicitação deve conter os seguintes campos no formato JSON:

{
	"coffee": "espresso",
	"price": 11.00,
	"sugar": false,
	"customer": "Thay"
}

O campo coffee representa o tipo de café que o cliente deseja, o campo price representa o preço do café, o campo sugar indica se o café deve conter açúcar ou não, e o campo customer representa o nome do cliente.

Consultar todos os pedidos

Para consultar todos os pedidos, faça uma solicitação HTTP GET para o seguinte endpoint:

GET /coffee/all

Consultar pedidos de um cliente

Para consultar os pedidos de um cliente, faça uma solicitação HTTP GET para o seguinte endpoint, substituindo <customer> pelo nome do cliente desejado:

GET /search_customer/<customer>
